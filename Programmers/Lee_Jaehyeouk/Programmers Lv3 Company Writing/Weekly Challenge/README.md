## Problem1

문제: 퍼즐 채우기 (https://programmers.co.kr/learn/courses/30/lessons/84021)

### Solution

Step 1. 퍼즐을 채웠을 때 주변에 빈칸이 있으면 안된다 = 딱 맞는 퍼즐을 넣어야 한다.

Step 2. 보드내에 블록이 들어가야할 위치를 찾는다.

Step 3. 넣어야 할 퍼즐을 4번 회전시킨 값을 구한다.

Step 4. 맞는 조각이 있으면 +1 을 해준다.
